		Web Crawler application using Jsoup library
---------------------------------------------------------

Running application
-------------------
1. Import project to eclipse
2. Add libs to Buildpath
3. Edit Run configurations to add src/main/config into classpath
4. Run as Java Application
5. Application starts and will prompt for url to enter.
6. Check for WebCrawlerResult.xml in project root directory
7. Check for logs in path project root directory --> logs --> webcrawlerpoc.log

Done..
